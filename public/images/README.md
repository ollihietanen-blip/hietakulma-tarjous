# Kuvat-kansio

Tämä kansio on tarkoitettu projektissa käytettäville kuville, kuten:

- Hietakulman logo
- Muut brändikuvat
- Projektin kuvat
- Muut staattiset kuvatiedostot

## Käyttö

Kuvat tässä kansiossa ovat saatavilla suoraan URL:lla:

```html
<img src="/images/hietakulma-logo.png" alt="Hietakulma Logo" />
```

Tai React-komponenteissa:

```tsx
<img src="/images/hietakulma-logo.png" alt="Hietakulma Logo" />
```

## Tuetut tiedostomuodot

- PNG
- JPG/JPEG
- SVG
- GIF
- WebP

## Esimerkki

Lisää Hietakulman logo tähän kansioon nimellä `hietakulma-logo.png` ja käytä sitä sitten koodissa:

```tsx
<img src="/images/hietakulma-logo.png" alt="Hietakulma" className="h-8" />
```
